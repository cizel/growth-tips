## 实践思路

> 参考陈皓《程序员练级攻略的正确打开方式》 

- `实验室` (写一些实验性的程序验证或知识点，出了问题可以进行修复)
- `工作室` (以完整的软件功能为主，可以让用户来用的东西)
- `工厂` (工厂是有相关的工业标准，有一整套的规范和标准)
